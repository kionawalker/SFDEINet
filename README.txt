事前準備
txtフォルダ内に入力画像のディレクトリまでのパスを記述したtextファイルを配置

Usage
①カレントディレクトリが4inputsとなるように移動

②source activate /venv/USFで仮想実行環境をactivate

③Two_times_two_in.pyを引数を指定して実行
第一引数：事前に準備したtextファイルまでのパス
第二引数：どの観測方向のデータで学習させたモデルを用いるか指定(55, 65, 75, 85のいずれか)
第三引数：どのサブセットを除いて学習させたモデルを用いるか指定(0, 1, 2, 3, 4のいずれか)
オプション引数-gpu GPUを利用する場合は -gpu 1と指定
